# Gemini 카드뉴스 생성 최적화 구조
# 메타(인스타/페북) 오리지널 보너스 프로그램 최적화 템플릿

## 카드뉴스 5장 구조 (슬라이드형)

### 슬라이드 1 — 훅 (Hook)
**목적**: 첫 0.3초 안에 스크롤을 멈추게 하는 강력한 페인포인트 자극
**포맷**:
- 상단: 타겟 키워드 강조 문구 (20자 이내)
- 중앙: 충격적 수치 또는 반전 질문
- 하단: "다음 장에서 해결법 공개" 예고 문구
**이미지 프롬프트 구조**: `[배경색] flat vector illustration, [주제 오브젝트], bold typography, high contrast, no text`

### 슬라이드 2 — 문제 정의 (Problem)
**목적**: 독자가 "맞아, 나 이거 힘든데" 공감 유발
**포맷**:
- Before 상황 3줄 불릿
- 감정적 언어 사용 (불안, 손해, 시간 낭비)
**이미지 프롬프트 구조**: `frustrated person, minimal style, muted colors, single character`

### 슬라이드 3 — 해결책 제시 (Solution)
**목적**: 무료 유틸리티 툴 소개 및 신뢰 구축
**포맷**:
- 툴 이름 + 핵심 기능 1줄 요약
- "무료", "즉시", "클릭 한 번" 키워드 포함
- UI 스크린샷 또는 결과물 예시
**이미지 프롬프트 구조**: `clean UI mockup, browser window, [툴 이름], light theme, professional`

### 슬라이드 4 — 사용법 (How-to)
**목적**: 3단계 이내로 사용법 압축, 진입 장벽 제거
**포맷**:
- Step 1 / Step 2 / Step 3 아이콘 번호형
- 각 단계 10자 이내 동사형 문장
**이미지 프롬프트 구조**: `step by step infographic, numbered icons, flat design, blue accent`

### 슬라이드 5 — CTA (리다이렉션 트리거)
**목적**: Cloudflare 정적 웹으로 전환 유도 → 2차 애드센스 수익 발생
**포맷**:
- "지금 바로 무료로 사용하기 →" 버튼형 문구
- 프로필 링크 클릭 유도 문구: "프로필 링크에서 바로 접속"
- 브랜드 로고 + 도메인 주소 명시
**이미지 프롬프트 구조**: `call-to-action card, bright orange button, arrow icon, white background, brand logo`

```
✅ [혜택 3]

#[타겟키워드] #무료툴 #[카테고리] #생산성 #꿀팁
```

## 해시태그 전략
- 1~3개: 메가 태그 (100만+ 게시물)
- 4~6개: 미드 태그 (10만~100만)
- 7~10개: 니치 태그 (1만~10만) ← 알고리즘 도달률 최고
