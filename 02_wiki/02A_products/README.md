# 02A_products — 유틸리티 웹 사양서 Vault

> Cloudflare Pages로 배포된 HTML/JS 단일 유틸리티 웹의 **완성 사양서**를 보관합니다.  
> 에이전트(Codex)가 빌드 완료 후 이 폴더에 사양서를 자동 저장합니다.

## 사양서 파일 네이밍 규칙
```
{키워드}-{yyyy-mm-dd}.md
예) compound-interest-calc-2026-06-29.md
```

## 사양서 필수 포함 항목
1. **타겟 키워드** (메인 1개 + LSI 5개)
2. **application/ld+json** 구조화 데이터 스펙
3. **AdSense 배치 영역** 3곳 좌표 명세
4. **계산 로직** 수도코드 (테스트 케이스 포함)
5. **배포 URL** (Cloudflare Pages)
6. **성과 기록** (일일 클릭, 애드센스 수익)
